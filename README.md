# Basic Chatbot using Python 🤖

This project is a simple rule-based chatbot created using Python as part of the **CodeAlpha Python Programming Internship**.

## 📌 Project Description
The chatbot responds to basic user inputs with predefined replies using conditional statements and loops.

## 🚀 Features
- Responds to greetings like **hello**
- Replies to questions like **how are you**
- Exits when the user types **bye**
- Beginner-friendly Python project

## 🛠️ Technologies Used
- Python
- if-elif conditions
- while loop
- input/output functions

## 💻 Sample Interaction
```bash
You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: bye
Bot: Goodbye!
```
